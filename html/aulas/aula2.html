Introdução ao JavaScript

JavaScript ≠ Java

Diferenças

  • JavaScript é uma linguagem de scripts OOP, ao passo que Java é uma linguagem de programação OOP.
  • O código JavaScript é executado apenas em um browser, ao passo que Java cria aplicações executadas em uma máquina virtual ou em um browser.
  • Os códigos JavaScript estão totalmente em texto, ao passo que o código Java precisa ser compilado.
  • Eles requerem plug-ins diferentes.

Maneiras de se utilizar o JavaScript

  • Dentro da tag <script>
  • De um arquivo externo
